Practice

Practical exercises for everyday professional skills.

Practice exercises will help learners apply skills through simple tasks, examples, and self-check guidance. These exercises are designed to run without storing user responses.

Planned exercise areas

These sections are being prepared. The goal is to keep practice lightweight, useful, and privacy-conscious.

01

AI prompting practice

Improve weak prompts, compare examples, and practise giving better context to AI tools.

02

Communication practice

Rewrite emails, improve updates, and practise clearer workplace messages.

03

Basic Excel practice

Work through basic formula, table, and data interpretation scenarios.

04

Interview answer practice

Practise structuring responses to common interview questions using clear examples and simple rubrics.

05

Chart reading practice

Practise reading simple charts, identifying trends, and explaining what the data means.

Privacy

Practice without unnecessary storage.

Practice exercises are intended to run in the browser without storing learner responses, unless a future feature clearly explains otherwise and asks for consent.
